## 🚀 Особенности

- **Оптимизация:** Автоматическая конвертация изображений в формат `WebP`.
- **Производительность:** Использование `woff2` для шрифтов и оптимизированные SVG-спрайты.
- **Адаптивность:** Полностью адаптивная верстка под различные устройства.
- **Инструменты:** Построен на базе Vite с кастомными плагинами для сборки.

## 🛠 Технологии

- HTML5 / SCSS
- JavaScript (ES6+)
- Vite
- Sharp (для обработки изображений)
- SVG Sprite generation

## 📂 Структура проекта

- `src/` — исходный код проекта.
- `docs/` — скомпилированная версия сайта (используется для деплоя на GitHub Pages).
- `public/` — статические ресурсы.

## 📦 Как запустить локально

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2. Клонируйте репозиторий:
   `git clone https://github.com/Qooler-667/Game-landing.git`
3. Установите зависимости:
   `npm install`
4. Запустите проект в режиме разработки:
   `npm run dev`
5. Соберите проект для публикации:
   `npm run build`
